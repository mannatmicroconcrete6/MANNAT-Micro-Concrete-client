# MANNAT Micro Concrete - Client

Welcome to the client-side repository for **MANNAT Micro Concrete**. This is a modern, high-performance web application built using [Next.js](https://nextjs.org).

## 🚀 Overview

This repository contains the frontend implementation of the MANNAT Micro Concrete platform. It features a responsive design, sleek aesthetics, and optimized user experience for showcasing micro concrete products and services.

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: Tailwind CSS / Vanilla CSS
- **State Management**: React Context / Hooks
- **Icons**: Lucide-React / FontAwesome
- **Deployment**: Vercel / Netlify

## 📦 Project Structure

```bash
client/
├── public/          # Static assets (images, icons, etc.)
├── src/
│   ├── app/         # Next.js App Router (pages and layouts)
│   ├── components/  # Reusable UI components
│   ├── lib/          # Utilities and helper functions
│   └── styles/      # Global CSS and themes
├── next.config.mjs  # Next.js configuration
├── package.json     # Project dependencies
└── README.md        # This file
```

## 🚥 Getting Started

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hanikumar0/MANNAT-Micro-Concrete-client.git
   ```

2. Navigate to the project directory:
   ```bash
   cd MANNAT-Micro-Concrete-client
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
